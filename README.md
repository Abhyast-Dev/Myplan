# My Plan

GitHub Pages-ready syllabus tracker.

## Files

- `index.html` - main app
- `css/styles.css` - styling
- `js/parser.js` - PDF and JSON parsing
- `js/app.js` - UI, progress, reset, export logic
- `logo.png` - logo placeholder
- `favicon.png` - favicon placeholder

## How to use

1. Upload the folder contents to a GitHub repository.
2. Enable GitHub Pages for the repository.
3. Open the Pages URL.
4. Enter learner name and upload a text-based syllabus PDF or import JSON.

The app stores data in browser localStorage. No backend is required.
